Module 1: Basic Use of R

Introduction to R and RStudio

R is a powerful programming language and environment specifically designed for statistical computing and graphics. It was created by Ross Ihaka and Robert Gentleman at the University of Auckland, New Zealand, and is now developed by the R Development Core Team. R is widely used among statisticians, data analysts, researchers, and data scientists for data analysis and developing statistical software.

Why Learn R?

  1. Open Source: R is free and open-source, making it accessible to everyone.
  2. Comprehensive Statistical Analysis: R offers a wide range of statistical techniques, from basic statistics to advanced machine learning algorithms.
  3. Excellent Visualization Capabilities: R provides powerful tools for creating high-quality graphics and visualizations.
  4. Large Community and Support: R has a large and active community that contributes to its development and provides support.
  5. Extensibility: R can be extended through packages, which are collections of R functions, data, and compiled code.
  6. Integration: R can be integrated with other languages and tools, such as Python, SQL, and C++.

Installing R and RStudio

Installing R

R can be downloaded from the Comprehensive R Archive Network (CRAN) website: https://cran.r-project.org/

For Windows:

  1. Go to https://cran.r-project.org/
  2. Click on "Download R for Windows"
  3. Click on "base"
  4. Click on the download link for the latest version
  5. Run the downloaded installer and follow the installation instructions

For Mac:

  1. Go to https://cran.r-project.org/
  2. Click on "Download R for macOS"
  3. Click on the download link for the latest version
  4. Run the downloaded installer and follow the installation instructions

For Linux:

For Ubuntu/Debian-based systems, you can install R using the following commands:

sudo apt-get update
sudo apt-get install r-base r-base-dev

For Red Hat/Fedora-based systems:

sudo yum install R

Installing RStudio

RStudio is an integrated development environment (IDE) for R that makes working with R much easier and more productive. It provides a user-friendly interface with many features that enhance the R programming experience.

  1. Go to https://www.rstudio.com/products/rstudio/download/
  2. Download the free RStudio Desktop version for your operating system
  3. Run the downloaded installer and follow the installation instructions

The RStudio Interface

RStudio has a clean, intuitive interface divided into four main panes:

  1. Source Editor (top-left): Where you write and edit your R scripts
  2. Console (bottom-left): Where you can run R commands and see their output
  3. Environment/History (top-right): Shows the objects in your workspace and command history
  4. Files/Plots/Packages/Help (bottom-right): Shows files in your working directory, plots, installed packages, and help documentation

Customizing RStudio

You can customize the appearance and behavior of RStudio through the Tools > Global Options menu. Some useful customizations include:

  • Changing the color theme (Tools > Global Options > Appearance)
  • Setting the default working directory (Tools > Global Options > General)
  • Configuring code completion (Tools > Global Options > Code)

Basic R Syntax and Operations

R as a Calculator

R can be used as a simple calculator:

# Addition
5 + 3  # Result: 8

# Subtraction
10 - 4  # Result: 6

# Multiplication
6 * 7  # Result: 42

# Division
15 / 3  # Result: 5

# Exponentiation
2^3  # Result: 8

# Modulo (remainder after division)
17 %% 5  # Result: 2

# Integer division
17 %/% 5  # Result: 3

Variables and Assignment

You can store values in variables using the assignment operator <- (or =):

# Assigning a value to a variable
x <- 10
y = 5  # Alternative assignment operator

# Printing the value of a variable
x  # Result: 10

# Performing operations with variables
z <- x + y  # z will be 15

Basic Functions

R has many built-in functions:

# Square root
sqrt(25)  # Result: 5

# Absolute value
abs(-7)  # Result: 7

# Rounding
round(3.14159, 2)  # Result: 3.14

# Natural logarithm
log(10)  # Result: 2.302585

# Exponential
exp(1)  # Result: 2.718282 (e)

# Sum
sum(1, 2, 3, 4, 5)  # Result: 15

# Mean
mean(c(1, 2, 3, 4, 5))  # Result: 3

Comments

Comments in R start with the # symbol:

# This is a comment
x <- 10  # This is also a comment

Data Types and Structures in R

Basic Data Types

  1. Numeric: Regular decimal numbers 
    x <- 10.5
class(x)  # Result: "numeric"
  2. Integer: Whole numbers 
    y <- 10L  # The 'L' suffix creates an integer
class(y)  # Result: "integer"
  3. Character: Text or string values 
    name <- "John"
class(name)  # Result: "character"
  4. Logical: Boolean values (TRUE or FALSE) 
    is_student <- TRUE
class(is_student)  # Result: "logical"
  5. Complex: Complex numbers 
    z <- 3 + 2i
class(z)  # Result: "complex"

Data Structures

  1. Vectors: One-dimensional arrays that can hold elements of the same type 
    # Creating a numeric vector
numbers <- c(1, 2, 3, 4, 5)

# Creating a character vector
fruits <- c("apple", "banana", "orange")

# Accessing elements of a vector
numbers[1]  # Result: 1 (R indexing starts at 1, not 0)
fruits[2]  # Result: "banana"

# Vector operations
numbers * 2  # Result: 2 4 6 8 10
numbers + c(10, 20, 30, 40, 50)  # Result: 11 22 33 44 55
  2. Matrices: Two-dimensional arrays with elements of the same type 
    # Creating a matrix
mat <- matrix(1:9, nrow = 3, ncol = 3)
mat
#      [,1] [,2] [,3]
# [1,]    1    4    7
# [2,]    2    5    8
# [3,]    3    6    9

# Accessing elements of a matrix
mat[1, 2]  # Result: 4 (element in row 1, column 2)
mat[2, ]   # Result: 2 5 8 (entire second row)
mat[, 3]   # Result: 7 8 9 (entire third column)
  3. Lists: Ordered collections that can contain elements of different types 
    # Creating a list
person <- list(name = "John", age = 30, is_student = FALSE, scores = c(85, 90, 78))

# Accessing elements of a list
person$name  # Result: "John"
person[[1]]  # Result: "John"
person$scores[2]  # Result: 90
  4. Data Frames: Table-like structures where columns can be of different types 
    # Creating a data frame
students <- data.frame(
  name = c("John", "Alice", "Bob"),
  age = c(22, 25, 23),
  gpa = c(3.5, 3.9, 3.2)
)

students
#    name age  gpa
# 1  John  22  3.5
# 2 Alice  25  3.9
# 3   Bob  23  3.2

# Accessing elements of a data frame
students$name  # Result: "John" "Alice" "Bob"
students[1, ]  # Result: First row
students[, "age"]  # Result: 22 25 23
students[2, 3]  # Result: 3.9 (element in row 2, column 3)
  5. Factors: Used to represent categorical data 
    # Creating a factor
gender <- factor(c("male", "female", "male", "female", "male"))

gender
# [1] male   female male   female male  
# Levels: female male

# Getting the levels of a factor
levels(gender)  # Result: "female" "male"

Importing and Exporting Data

Working Directory

Before importing or exporting data, it's important to know your working directory:

# Get the current working directory
getwd()

# Set the working directory
setwd("/path/to/your/directory")  # Replace with your actual path

Importing Data

  1. CSV Files: 
    # Reading a CSV file
data <- read.csv("data.csv")

# Reading a CSV file with specific options
data <- read.csv("data.csv", header = TRUE, sep = ",", stringsAsFactors = FALSE)
  2. Excel Files:

    To read Excel files, you need to install the readxl package:

    # Install the readxl package (only need to do this once)
install.packages("readxl")

# Load the readxl package
library(readxl)

# Reading an Excel file
data <- read_excel("data.xlsx", sheet = 1)
  3. Text Files: 
    # Reading a tab-delimited text file
data <- read.table("data.txt", header = TRUE, sep = "\t")
  4. R Data Files: 
    # Loading an R data file
load("data.RData")

Exporting Data

  1. CSV Files: 
    # Writing a data frame to a CSV file
write.csv(data, "output.csv", row.names = FALSE)
  2. Excel Files:

    To write Excel files, you need to install the writexl package:

    # Install the writexl package (only need to do this once)
install.packages("writexl")

# Load the writexl package
library(writexl)

# Writing a data frame to an Excel file
write_xlsx(data, "output.xlsx")
  3. Text Files: 
    # Writing a data frame to a tab-delimited text file
write.table(data, "output.txt", sep = "\t", row.names = FALSE)
  4. R Data Files: 
    # Saving R objects to an R data file
save(data, file = "output.RData")

Basic Data Manipulation

Subsetting Data

# Creating a sample data frame
df <- data.frame(
  id = 1:5,
  name = c("John", "Alice", "Bob", "Carol", "David"),
  age = c(25, 30, 22, 28, 35),
  score = c(85, 92, 78, 88, 95)
)

# Subsetting rows based on a condition
young_students <- df[df$age < 30, ]

# Subsetting columns
names_ages <- df[, c("name", "age")]

# Subsetting both rows and columns
selected_data <- df[df$score > 80, c("id", "name", "score")]

Adding and Modifying Columns

# Adding a new column
df$grade <- c("B", "A", "C", "B", "A")

# Creating a column based on existing columns
df$pass <- df$score >= 80

# Modifying an existing column
df$age <- df$age + 1  # Incrementing all ages by 1

Sorting Data

# Sorting by a single column (ascending)
df_sorted <- df[order(df$age), ]

# Sorting by a single column (descending)
df_sorted_desc <- df[order(-df$score), ]

# Sorting by multiple columns
df_sorted_multi <- df[order(df$grade, df$age), ]

Basic Summary Statistics

# Summary of the entire data frame
summary(df)

# Summary of a specific column
summary(df$age)

# Mean, median, min, max of a column
mean(df$score)
median(df$score)
min(df$score)
max(df$score)
range(df$score)

Basic Data Visualization

Base R Plotting

  1. Scatter Plot: 
    # Creating a scatter plot
plot(df$age, df$score, 
     main = "Age vs. Score",
     xlab = "Age",
     ylab = "Score",
     pch = 19,  # Point type
     col = "blue")
  2. Line Plot: 
    # Creating a line plot
x <- 1:10
y <- x^2
plot(x, y, 
     type = "l",  # 'l' for line
     main = "Line Plot Example",
     xlab = "X",
     ylab = "Y",
     col = "red")
  3. Bar Plot: 
    # Creating a bar plot
counts <- table(df$grade)
barplot(counts, 
        main = "Grade Distribution",
        xlab = "Grade",
        ylab = "Frequency",
        col = c("red", "green", "blue"))
  4. Histogram: 
    # Creating a histogram
hist(df$score, 
     main = "Score Distribution",
     xlab = "Score",
     ylab = "Frequency",
     col = "lightblue",
     breaks = 10)
  5. Box Plot: 
    # Creating a box plot
boxplot(df$score ~ df$grade, 
        main = "Score by Grade",
        xlab = "Grade",
        ylab = "Score",
        col = c("red", "green", "blue"))

Saving Plots

# Opening a PNG device
png("plot.png", width = 800, height = 600)

Introduction to R Packages

R packages are collections of R functions, data, and compiled code that extend the functionality of R.

Installing Packages

# Installing a single package
install.packages("ggplot2")

# Installing multiple packages
install.packages(c("dplyr", "tidyr", "readr"))

Loading Packages

# Loading a package
library(ggplot2)

# Checking which packages are loaded
search()

Popular R Packages

  1. ggplot2: For creating elegant data visualizations
  2. dplyr: For data manipulation
  3. tidyr: For tidying data
  4. readr: For reading rectangular data
  5. readxl: For reading Excel files
  6. lubridate: For working with dates and times

Example: Using ggplot2 for Visualization

# Install and load ggplot2
install.packages("ggplot2")
library(ggplot2)

# Creating a scatter plot with ggplot2
ggplot(df, aes(x = age, y = score, color = grade)) +
  geom_point(size = 3) +

       x = "Age",
       y = "Score") +
  theme_minimal()

Question 1:

Create a vector containing the numbers 5, 10, 15, 20, and 25. Then, calculate the mean and standard deviation of this vector.

Answer 1:

# Create the vector
my_vector <- c(5, 10, 15, 20, 25)

# Calculate the mean
mean_value <- mean(my_vector)
print(mean_value)  # Result: 15

# Calculate the standard deviation
sd_value <- sd(my_vector)
print(sd_value)  # Result: 8.660254

Question 4:

Install and load the "dplyr" package. Then, create a data frame with columns for student names, test scores, and grades. Use dplyr functions to arrange the data frame in descending order of test scores.

students <- data.frame(
